# @stack-test/cli

Command-line interface for StackTest, a provider-agnostic infrastructure testing framework.

## Install

```bash
npm install --save-dev @stack-test/cli
```

## Usage

```bash
npx stacktest --help
npx stacktest plan
npx stacktest run --provider fake
```
